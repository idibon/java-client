java-client
===========

Idibon API Java client

To generate jar file and javadocs, install maven

Generate jar:
1. In api folder:
  mvn clean install
2. Generated jar: ./target/api-1.0-jar-with-dependencies.jar

Generate javadocs:
1. In api folder:
  mvn javadoc:javadoc
2. Generated docs: ./target/site/apidocs/
